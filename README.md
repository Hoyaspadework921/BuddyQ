## 📋 What is BuddyQ?

BuddyQ is a local AI voice assistant for Windows. It runs completely on your computer without needing the internet or cloud services. You can control your PC using your voice. BuddyQ listens and understands your commands with privacy because everything happens on your own hardware.

It uses three main technologies:
- **faster-whisper** for speech recognition (turning your voice into text)
- **Piper TTS** for speech output (turning text back into voice)
- An OpenAI-compatible local large language model (either LM Studio or llama.cpp)

This setup lets you interact with your PC using natural language to do many tasks like:
- Searching the web
- Setting timers
- Taking notes
- Checking the weather
- Using screen and vision tools

You do not need to sign up for anything, supply API keys, or rely on external services.

## 🎤 How to Use BuddyQ

### Activate Voice Commands

- To start talking, activate the "listen" button in the app or press the assigned hotkey (check settings).
- Speak your command in a clear voice.
- BuddyQ uses built-in speech recognition and language models to understand your request.

### Typical Commands

- **"Search for [topic]"** – Performs a web search and reads summaries.
- **"Set a timer for 10 minutes"** – Creates a countdown timer.
- **"Add a note: Buy milk"** – Saves a quick note.
- **"What's the weather today?"** – Reads local weather information.
- **"Read this screen"** – Uses vision tools to describe screen contents.

## 🧰 Technical Details (Optional Interest)

BuddyQ integrates local AI tools to maximize speed and privacy:

- **faster-whisper**: A fast speech-to-text engine using open-source Whisper neural networks.
- **Piper TTS**: Text-to-speech engine delivering natural voice output.
- **LM Studio / llama.cpp**: Local large language models that respond like OpenAI's ChatGPT but run fully offline.

This design avoids any cloud connection by default. The web search feature uses local queries without uploading voice or personal data.
